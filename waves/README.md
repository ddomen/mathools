[@mathools](https://github.com/ddomen/mathools) waves module, a library of waves functions

## Contains
* **Parametric** `parametric(number, isDegree)` => constrain a `number` to stay between `0` and `360` ( or `0` and `2Ï€`)

* **Square** `square(x)` => squared wave

* **Triangle** `triangle(x)` => triangle wave

* **Sawtooth** `sawtooth(x)` => sawtooth wave

* **Rounded** `rounded(x)` => rounded sin wave

* **Noise** `noise(x)` => randomic wave
